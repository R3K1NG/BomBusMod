# Install
```
cd $HOME
git clone https://github.com/Goblinor/BomBusMod.git
cd BomBusMod
chmod +x install.sh
chmod +x launch.sh
chmod +x tg
./install.sh
./launch.sh
# Enter a phone number & confirmation code.
```
