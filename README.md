# [BomBusMod](https://github.com/R3K1NG/BomBusMod)
*TG-CLI based broadcasting bot!*
# Install
```
cd $HOME
git clone https://github.com/R3K1NG/BomBusMod
cd BomBusMod
chmod +x install.sh
chmod +x launch.sh
chmod +x tg
./install.sh
./launch.sh
# Enter a phone number & confirmation code.
```
Create a bot!
